# Описание файлов в директории `data`

В этом документе описаны файлы, которые используются в проекте для подготовки выборки по авиабилетам и сопутствующим справочникам.

## 1. API

### 1.1 tickets.csv

Описание: основной датасет с ценами на авиабилеты из Москвы (`MOW`) в российские города из датасета `cities_prepared.csv`. Для каждого направления выполнялись запросы на разных маркетах и в нескольких валютах.

Источник: Travelpayouts / Aviasales Data API.

Получено с помощью вызова ручки: `http://api.travelpayouts.com/v2/prices/month-matrix`

Согласно логике в [parser/destonation.py](/SMADIMO-GP-2/parser/destonation.py):

- месяц запроса: `2026-06-01`
- направление отправления: `MOW`
- выбираются прямые перелёты: `direct=true`
- выбираются билеты в одну сторону: `one_way=true`
- перебираются маркеты из `market_prepared.csv`
- для каждого маркета перебираются валюты: дефолтная валюта маркета + набор `rub`, `usd`, `eur`, `cny`
- курс валют подставляется из файла `currency.json`

| Переменная | Тип данных | Описание |
|------------|------------|----------|
| depart_date | String | Дата вылета |
| origin | String | IATA-код точки отправления |
| destination | String | IATA-код точки назначения |
| gate | String | Агент или площадка, где найден билет |
| trip_class | String / Integer | Класс перелёта |
| value | Float | Стоимость билета в указанной валюте |
| duration | Float | Длительность перелёта |
| distance | Float | Расстояние между пунктами |
| market | String | Маркет Aviasales, с которым выполнялся запрос |
| currency | String | Валюта цены |
| rate | Float | Курс валюты к рублю из `currency.json` |

### 1.2 cities-aviasales.csv

Описание: справочник российских городов, полученный из API Aviasales. Используется как исходный список городов с координатами, часовыми поясами и признаком наличия аэропорта.

Источник: ручка `http://api.travelpayouts.com/data/ru/cities.json`

Файл сформирован скриптом [parser/get_russian_cities.py](/SMADIMO-GP-2/parser/get_russian_cities.py). В итоговый CSV попадают только города, у которых `country_code == "RU"`.

| Переменная | Тип данных | Описание |
|------------|------------|----------|
| name | String | Название города на русском языке |
| name_en | String | Название города на английском языке |
| code | String | IATA-код города |
| lat | Float | Широта |
| lon | Float | Долгота |
| has_flightable_airport | Boolean | Есть ли в городе аэропорт, пригодный для перелётов |
| time_zone | String | Часовой пояс города |

### 1.3 currency.json

Описание: справочник валютных курсов, используемый для приведения цен из разных валют к общей шкале.

Источник: ручка `http://yasen.aviasales.ru/adaptors/currency.json`

Файл сформирован скриптом [parser/currency.py](/SMADIMO-GP-2/parser/currency.py)

Формат файла: JSON-объект вида `{"usd": ..., "eur": ..., "cny": ...}`, где ключом является код валюты, а значением курс к рублю.

Использование в проекте:

- файл читается в [parser/destonation.py](/SMADIMO-GP-2/parser/destonation.py)
- по полю `currency` в `tickets.csv` подставляется значение `rate`

### 1.4 airline_info.json

Описание: дополнительный JSON-файл со сведениями об авиакомпаниях.

Источник: ручка `http://yasen.aviasales.ru/adaptors/currency.json`

Файл сформирован скриптом [parser/airlines.py](/SMADIMO-GP-2/parser/airlines.py)

Назначение в проекте:

- хранение справочной информации по авиакомпаниям
- сопоставление кодов и наименований при дальнейшей обработке

## 2. Файлы Википедии / другие источники

### 2.1 wiki-ru-cities.csv

Описание: список городов России, извлечённый со страницы Википедии про города России. Используется как внешний источник для последующего сопоставления с городами из Aviasales.

Источник: страница Википедии `Города России`

Файл сформирован скриптом [parser/wiki_parser.py](/Users/leonidprokopev/projects/SMADIMO-GP-2/parser/wiki_parser.py), который:

- открывает страницу Википедии
- проходит по нескольким секциям с группировкой городов по численности населения
- извлекает названия городов из таблиц
- удаляет дубликаты

Структура файла:

| Переменная | Тип данных | Описание |
|------------|------------|----------|
| Unnamed: 0 | Integer | Индекс, записанный при сохранении CSV |
| city | String | Название города |

### 2.2 Список_действующих_авиакомпаний_России_—_Википедия.csv

Описание: список действующих авиакомпаний России.

Источник: Википедия.

Файл получен из Википедии просто руками.

Структура файла:

| Переменная | Тип данных | Описание |
|------------|------------|----------|
| Бренд | String | Наименование авиакомпании |
| ИНН | String | ИНН организации |
| ИАТА | String | IATA-код авиакомпании, если указан |
| Код в России | String | Внутренний код авиакомпании в России |

## 4. Итоговые датасеты, полученные в результате работы

### 4.1 market_prepared.csv

Описание: сокращённая и подготовленная версия файла `market.xlsx - Sheet1.csv`, используемая в коде.

Назначение:

- содержит только те поля, которые нужны для генерации запросов к API
- используется в [parser/main.py](/SMADIMO-GP-2/parser/main.py) для построения словаря маркетов и их дефолтных валют

Структура файла:

| Переменная | Тип данных | Описание |
|------------|------------|----------|
| Маркет | String | Код маркета |
| Дефолтная валюта | String | Основная валюта маркета |

### 4.2 cities_prepared.csv

Описание: подготовленный итоговый список российских городов для запросов к API авиабилетов.

Файл получен как результат объединения и очистки данных из:

- `cities-aviasales.csv`
- `wiki-ru-cities.csv`

Назначение:

- использовать только релевантные российские города
- привести IATA-код к единому названию поля `IATA_code`
- добавить порядок или приоритет городов через поле `population_rating`

Структура файла:

| Переменная | Тип данных | Описание |
|------------|------------|----------|
| name | String | Название города на русском языке |
| name_en | String | Название города на английском языке |
| IATA_code | String | IATA-код города |
| lat | Float | Широта |
| lon | Float | Долгота |
| has_flightable_airport | Boolean | Наличие пригодного аэропорта |
| time_zone | String | Часовой пояс |
| population_rating | Integer | Порядковый рейтинг города по приоритету / населению |
